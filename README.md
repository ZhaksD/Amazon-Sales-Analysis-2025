# Amazon-Sales-Analysis-2025
Этот проект посвящен анализу продаж Amazon с целью выявления ключевых факторов дохода, популярных товаров и поведения покупателей.  В рамках проекта демонстрируется стандартный workflow аналитика данных:  - очистка данных - исследовательский анализ (EDA) - визуализация данных - создание интерактивного дашборда
