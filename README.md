# Amazon-Sales-Analysis-2025
Этот проект посвящен анализу продаж Amazon с целью выявления ключевых факторов дохода, популярных товаров и поведения покупателей.  В рамках проекта демонстрируется стандартный workflow аналитика данных:  - очистка данных - исследовательский анализ (EDA) - визуализация данных - создание интерактивного дашборда
Датасет содержит 250 записей транзакций Amazon, включая информацию о товарах, клиентах, способах оплаты и статусах заказов.

## Dataset Description / Описание данных

Датасет содержит следующие поля:

| Column | Description |
|------|-------------|
| Order ID | уникальный идентификатор заказа |
| Date | дата заказа |
| Product | название товара |
| Category | категория товара |
| Price | цена товара |
| Quantity | количество |
| Total Sales | общий доход по заказу |
| Customer Name | имя клиента |
| Customer Location | город клиента |
| Payment Method | способ оплаты |
| Status | статус заказа |

---

## Data Cleaning / Очистка данных

Очистка данных была выполнена с использованием Python (pandas):

- преобразование колонок Price и Total Sales в числовой формат
- преобразование колонки Date в формат datetime
- удаление дубликатов
- обработка пропущенных значений
- создание дополнительных временных признаков

После очистки данные были сохранены в Excel и использованы для построения дашборда в Power BI.

---

## Exploratory Data Analysis (EDA)

Был проведен исследовательский анализ данных для изучения структуры продаж.

Основные направления анализа:

- распределение продаж по категориям
- топ товаров по доходу
- динамика дохода по времени
- распределение заказов по статусам
- анализ способов оплаты

Для визуализации использовались библиотеки matplotlib и seaborn.

## Key Insights / Основные выводы

- Некоторые категории товаров приносят значительно больше дохода.
- Небольшое количество товаров формирует основную часть продаж.
- Большинство заказов успешно завершается.
- Наиболее популярные способы оплаты — банковские карты и онлайн-платежи PayPal.
- Доход показывает изменения во времени, что может свидетельствовать о сезонности.
